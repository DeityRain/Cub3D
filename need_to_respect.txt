The first raycasting step is:

Cast one ray for each vertical column of the screen

so for x = 0 to WIN_WIDTH - 1
Convert the screen column into a camera coordinate

cameraX = 2 * x / WIN_WIDTH - 1

Build the ray direction

rayDirX = dir_x + plane_x * cameraX
rayDirY = dir_y + plane_y * cameraX
So the very first raycasting goal is just:

take the player direction
take the camera plane
compute a ray direction for each column
